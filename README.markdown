## Install

### Unix
```
cd ~
git clone https://github.com/vais/dotfiles.git
dotfiles/sync
```

### Windows
```
cd %HOMEPATH%
git clone https://github.com/vais/dotfiles.git
dotfiles\sync
```

## Sync

### Unix
```
~/dotfiles/sync
```

### Windows
```
%HOMEPATH%\dotfiles\sync.bat
```
